## 💻 주요 기능
- 스터디 개설 및 가입
  - 원하는 언어, 인원수를 설정해 스터디 개설
  - 이미 개설된 스터디에 가입
- 스터디 진행
  - 데일리 문제 등록하기
  - 코드 작성 및 제출
  - 다른 사람의 풀이에 좋아요 표시 및 댓글
- 내 코드 관리
  - 출석체크  
  - 작성한 코드 모아보기
  - 내가 좋아요한 다른 사람 코드 모아보기
